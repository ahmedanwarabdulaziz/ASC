# Public Website and CMS | الموقع العام ولوحة المحتوى

## Goal | الهدف

Create a professional Arabic-first public website for Assiut Sporting Club, but not as the first full build. At the beginning, the public side will be a simple entry page with a button to enter the internal system.

في البداية الموقع العام سيكون صفحة بسيطة للدخول للسيستم، ثم لاحقا يتم بناء الموقع الاحترافي ولوحة المحتوى الاعلامي.

## Phase 0: Temporary Entry Page | صفحة الدخول المؤقتة

- Club name and basic identity | اسم النادي والهوية الاساسية
- Empty/simple public page | صفحة عامة بسيطة
- Enter System button | زر دخول السيستم
- Login route for internal users | مسار تسجيل دخول المستخدمين الداخليين
- Mobile-friendly layout | مناسبة للموبايل

This page exists so we can start the real operational system first without blocking on the full public website.

هذه الصفحة مؤقتة حتى نبدأ السيستم التشغيلي اولا بدون انتظار بناء الموقع العام بالكامل.

## Phase 1: Future Website Foundation | تأسيس الموقع لاحقا

- Home page | الصفحة الرئيسية
- About the club | عن النادي
- Sports and activities | الالعاب والانشطة
- News and announcements | الاخبار والاعلانات
- Facilities | المنشآت والملاعب
- Board of directors | مجلس الادارة
- Contact and location | التواصل والموقع

## Phase 2: CMS Access | صلاحيات المحتوى

- Media officer role | دور المسؤول الاعلامي
- Login-protected content dashboard | لوحة محتوى محمية بتسجيل الدخول
- Image upload and gallery management | رفع الصور وادارة المعارض
- News/article editor | محرر الاخبار
- Publish/draft workflow | نشر ومسودات
- Basic approval option for sensitive content | اختيار موافقة قبل النشر للمحتوى الحساس

## Phase 3: Dynamic Sports Sync | ربط الالعاب بالنظام

When a sport is added in the internal system, it should optionally appear on the public website.

عند اضافة لعبة في السيستم، يظهر اختيار لنشرها تلقائيا في الموقع.

Suggested fields:

- Arabic name | الاسم العربي
- English name | الاسم الانجليزي
- Description | الوصف
- Cover image | صورة رئيسية
- Coaches shown publicly | المدربين الظاهرين للعامة
- Training age groups | الفئات العمرية
- Public/private visibility | الظهور للعامة

## Phase 4: Professional Website Quality | جودة الموقع

- Fast mobile-first pages | صفحات سريعة على الموبايل
- Arabic RTL layout | تصميم عربي من اليمين لليسار
- English helper labels in admin where useful | تسميات انجليزية مساعدة في لوحة الادارة
- SEO basics | اساسيات الظهور في البحث
- Optimized images | صور محسنة
- Accessibility basics | اساسيات سهولة الاستخدام

## Key Permissions | الصلاحيات المهمة

- Website viewer | زائر الموقع
- Media officer | مسؤول اعلامي
- Content approver | معتمد محتوى
- Super admin | مدير عام للنظام

## Open Decisions | قرارات مفتوحة

- Should news publishing require approval or can the media officer publish directly?
- Should sports pages show subscription prices publicly?
- Should public booking be allowed later, or only authenticated booking?
