# نظام إدارة انتخابات نادي ASC

نظام بسيط وسريع لإدارة أعضاء فريق الانتخابات وتوزيعهم على الفرق.

## المميزات

- ✅ **صفحة تعريف**: صفحة ترحيبية لتقديم نفسك لأعضاء النادي
- ✅ **استيراد من Excel**: استيراد جميع الأعضاء من ملف Excel بسهولة
- ✅ **البحث بالعربية**: بحث متقدم يدعم اللغة العربية
- ✅ **إدارة الفرق**: إنشاء فرق وتوزيع الأعضاء عليها
- ✅ **متابعة الحالة**: تتبع حالة كل عضو (نشط، غير نشط، قيد الانتظار)
- ✅ **الملاحظات**: إضافة ملاحظات لكل عضو

## التقنيات المستخدمة

- **Next.js 14** - إطار عمل React
- **Firebase** - قاعدة البيانات والمصادقة
- **TypeScript** - للكتابة الآمنة
- **Tailwind CSS** - للتصميم
- **xlsx** - لقراءة ملفات Excel

## التثبيت والتشغيل

1. تثبيت المكتبات:
```bash
npm install
```

2. تشغيل المشروع في وضع التطوير:
```bash
npm run dev
```

3. افتح المتصفح على:
```
http://localhost:3000
```

## الاستخدام

### 1. استيراد الأعضاء من Excel

1. اذهب إلى `/dashboard`
2. في قسم "استيراد الأعضاء من Excel"، اختر ملف Excel الخاص بك
3. سيتم استيراد جميع البيانات تلقائياً

**ملاحظة**: يجب أن يحتوي ملف Excel على عمود يحتوي على أسماء الأعضاء (يمكن أن يكون اسم العمود: name، اسم، الاسم، إلخ)

### 2. إنشاء الفرق

1. اذهب إلى `/dashboard/teams`
2. اضغط على "إنشاء فريق جديد"
3. أدخل اسم الفريق واسم المرشح
4. اضغط "إنشاء الفريق"

### 3. توزيع الأعضاء على الفرق

1. اذهب إلى `/dashboard/teams/[team-id]`
2. اضغط على "تعيين عضو جديد"
3. اختر العضو من القائمة
4. اضغط "تعيين"

### 4. متابعة الأعضاء

1. من لوحة التحكم `/dashboard`، يمكنك:
   - البحث عن الأعضاء بالاسم، البريد الإلكتروني، أو الهاتف
   - تصفية الأعضاء حسب الفريق
   - عرض تفاصيل كل عضو وتعديل حالته وملاحظاته

## هيكل قاعدة البيانات

### Collection: `members`
```typescript
{
  id: string;
  name: string;              // الاسم الأصلي
  nameSearch: string;        // الاسم المعياري للبحث
  email?: string;
  phone?: string;
  teamId?: string;
  teamName?: string;
  status: 'active' | 'inactive' | 'pending';
  notes?: string;
  createdAt: Date;
  updatedAt: Date;
  // جميع أعمدة Excel الأخرى
}
```

### Collection: `teams`
```typescript
{
  id: string;
  name: string;
  nameSearch: string;
  candidateId: string;
  candidateName: string;
  members: string[];         // IDs of members
  createdAt: Date;
}
```

## المصادقة (قيد التطوير)

حالياً النظام متاح للجميع. سيتم إضافة نظام المصادقة مع الأدوار التالية:
- **Admin**: صلاحيات كاملة
- **Supervisor**: إشراف على الفرق
- **Team Leader**: إدارة فريق واحد

## ملاحظات مهمة

- النظام يستخدم Firebase Free Tier - تأكد من عدم تجاوز الحدود
- جميع البيانات محفوظة في Firestore
- البحث بالعربية يعمل عن طريق تطبيع النصوص (إزالة التشكيل، توحيد الأحرف)

## الدعم

للمساعدة أو الاستفسارات، يرجى التواصل مع فريق التطوير.
